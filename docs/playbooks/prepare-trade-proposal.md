# Prepare Trade Proposal

Use this before any trade execution request. This file is the canonical proposal template.

## Proposal format

- asset / pair
- venue or execution path
- thesis
- position size
- entry conditions
- exit or invalidation plan
- downside and operational risks
- fees, slippage, and liquidity concerns (when relevant)
- exact approval required

## Behavior rules

- be conservative
- do not blur recommendation with execution
- do not imply certainty

## Scope boundary

This playbook covers proposal content only.

Trading constraints are defined in [../policies/trading-guardrails.md](../policies/trading-guardrails.md).
